# E-commerce Book Store Data Analytics Project

## Project Overview
This project demonstrates an end-to-end data analytics workflow using e-commerce product data. Product information is scraped from a scraping-friendly e-commerce website, cleaned and analyzed using Excel, and visualized through an interactive Power BI dashboard to support data-driven decision-making.

---

## Tools & Technologies
- Python  
- BeautifulSoup  
- Requests  
- Pandas  
- Excel  
- Power BI  

---

## Data Collected
- Product Name  
- Price  
- Rating  
- Availability  

---

## Workflow
1. Scrape product data using Python  
2. Clean and structure the dataset  
3. Export data to Excel  
4. Create an interactive Power BI dashboard  

---

## Key Insights
- Pricing distribution across products  
- Relationship between ratings and price  
- Product availability overview  

---

## Outcome
The project provides clear insights into pricing trends, quality perception, and inventory status using interactive visualizations.
